Available placeholders

{% templatetag openvariable %}form_name{% templatetag closevariable %} {% templatetag openvariable %}submission_time{% templatetag closevariable %} {% templatetag openvariable %}ip_address{% templatetag closevariable %} {% templatetag openvariable %}fields_html{% templatetag closevariable %} {% templatetag openvariable %}fields_text{% templatetag closevariable %}

Per-field placeholders like {% templatetag openvariable %}name{% templatetag closevariable %} or {% templatetag openvariable %}email{% templatetag closevariable %} are also substituted from the submission payload.

Tips

  • ●Subject — supports placeholders too. Keep it short.
  • ●HTML body — rendered when the recipient’s mail client supports HTML.
  • ●Plain‑text body — optional fallback for plain‑text clients.
  • ●Mark a template as default to use it across all forms.
